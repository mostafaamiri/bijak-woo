ارسال کالا به صورت هوشمند به سرتاسر ایران به کمک باربری‌ها و اتصال مستقیم فروشگاه ووکامرس شما به سرویس بیجک.

== Description ==

افزونه **بیجک (ترابری هوشمند)** یک روش حمل‌ونقل جدید به ووکامرس اضافه می‌کند تا فروشگاه‌های اینترنتی بتوانند
سفارشات خود را به‌طور مستقیم در سامانه‌ی بیجک ثبت و مدیریت کنند.

**امکانات اصلی:**
* انتخاب شهر مقصد بیجک در صفحه تسویه‌حساب
* پشتیبانی از حالت **پیش‌کرایه** و **پس‌کرایه**
* محاسبه هزینه‌ی ارسال از API بیجک و نمایش به مشتری
* امکان تحویل درب منزل یا تحویل به باربری مقصد
* ثبت خودکار سفارش در سامانه بیجک پس از پرداخت
* نمایش وضعیت و کد پیگیری بیجک در پنل کاربر و مدیر فروشگاه

== Installation ==

1. افزونه را دانلود کرده و پوشه `bijak-woo` را در مسیر `wp-content/plugins` بارگذاری کنید.
2. افزونه را از بخش "افزونه‌ها" در پیشخوان وردپرس فعال کنید.
3. در تنظیمات افزونه، کلید API بیجک خود را وارد کنید.
4. شهر مبدأ و سایر گزینه‌ها را پیکربندی کنید.
5. در تنظیمات حمل‌ونقل ووکامرس، روش "ارسال کالا با بیجک" را فعال کنید.

== Frequently Asked Questions ==

= آیا نیاز به حساب کاربری بیجک دارم؟ =
بله، برای دریافت API Key باید در سامانه بیجک حساب کاربری داشته باشید.

= هزینه حمل چگونه محاسبه می‌شود؟ =
هزینه‌ها به‌صورت آنلاین از API بیجک دریافت می‌شوند و در صورتحساب مشتری نمایش داده می‌شوند.

= آیا می‌توانم همزمان چند روش حمل داشته باشم؟ =
بله، این افزونه به‌عنوان یک روش حمل جدید اضافه می‌شود و شما می‌توانید روش‌های دیگر را نیز فعال نگه دارید.
